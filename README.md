# 🛒 E-Commerce Data Analysis & Seller Performance Dashboard

## 📊 Project Overview

This project is an end-to-end e-commerce data analysis project where data was cleaned and processed using Python and visualized using Power BI to generate business insights related to sales, customer satisfaction, seller performance, and delivery efficiency.

---

## 🛠 Tools & Technologies Used

* Python (Pandas)
* Excel
* Power BI
* Power Query
* DAX
* GitHub

---

## 📁 Project Files

* `python_data_cleaning.py` → Data cleaning and data processing using Python
* `final_dataset.xlsx` → Final cleaned dataset used in Power BI
* `dashboard.pbix` → Power BI dashboard file
* `dashboard.png` → Dashboard screenshot
* `README.md` → Project documentation

---

## 🔄 Project Workflow

1. Data Collection (Excel datasets)
2. Data Cleaning & Processing (Python)
3. Data Transformation (Power Query)
4. Data Modeling (Power BI)
5. Data Visualization (Power BI Dashboard)
6. Business Insights Generation

---

## 📈 Key Business Insights

* Top Selling Product Categories
* Revenue by State
* Payment Method Analysis
* Customer Satisfaction vs Delivery Delay
* Delivery Efficiency (On-Time Delivery %)
* Average Delivery Time
* Total Revenue & Total Orders

---

## 📊 Dashboard Features

* KPI Cards (Total Revenue, Total Orders, Avg Delivery Days, On-Time Delivery %)
* Filled Map (Revenue by State)
* Donut Chart (Payment Method Distribution)
* Scatter Plot (Customer Satisfaction vs Delivery Delay)
* Bar Chart (Top Selling Categories)
* Gauge Chart (Delivery Efficiency)
---

## 🐍 Data Cleaning Steps (Python)

* Merged multiple tables (Orders, Order Items, Products, Sellers, Payments)
* Created Delivery Delay column
* Created Total Order Value column
* Extracted Month and Year from Order Date
* Handled missing values
* Exported final cleaned dataset to Excel

---

## 🎯 Project Outcome

This dashboard helps businesses to:

* Track seller performance
* Monitor delivery efficiency
* Analyze customer satisfaction
* Identify top revenue generating products
* Understand payment trends
